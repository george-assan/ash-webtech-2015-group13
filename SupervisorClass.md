This class extends the adb class that connects to the database.

Function add\_supervisor($name, $area)
This is a method that adds a supervisor to the supervisor database using parameters name and area.
Function get\_supervisor($name)
This is a method that retrieves a supervisor from the database using the parameter $name.
Function update\_supervisor($id, $name, $area)
This is a method that edits the values of a supervisor in the database. It takes parameters   $id, $name and $area .
Function delete\_supervisor($id)
This is a method that removes a supervisor from the database using the supervisor id from the parameter $id.
Function get\_all\_supervisors()
This is a method that retrieves all supervisors in the database.
